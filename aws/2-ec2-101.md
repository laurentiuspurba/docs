## EC2 Instance Type ##

![Alt text](images/EC2InstanceType.png?raw=true "EC2 Instance Type")

How I remember them in 2017:
* **D** Density
* **R** RAM
* **M** - main choice for general purpose
* **C** Compute
* **G** Graphic
* **I** IOPS
* **F** FPGA
* **T** cheap general purpose (think T2 Micro)
* **P** Graphics (thin Pics)
* **X** Extreme Memory
